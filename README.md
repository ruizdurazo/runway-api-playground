# Runway API Playground

This is a playground for the Runway API. It will be a simple web app that allows users to create and edit prompts that will generate images and videos.

## Technical Requirements

- [ ] create a new next.js project with typescript
- [ ] setup .gitignore
- [ ] setup env variables for supabase
- [ ] setup login page
- [ ] setup dashboard layout (with sidebar)
- [ ] setup main page (chat page)
- [ ] setup user media gallery page
- [ ] setup user settings page

## Requirements

- [ ] be able to sign up and create a new account with supabase
- [ ] be able to login
- [ ] be able to logout
- [ ] be able to change the user's name
- [ ] be able to set the user's runway api key
- [ ] be able to see a media gallery of all the images and videos uploaded by the user
- [ ] be able to create a new chat and prompt that will generate a video or image
- [ ] be able to edit and re-run a prompt with the same chat
- [ ] be able to delete a prompt from a chat and all the media generated from it
- [ ] be able to see a list/history of chats

## Tools

- Node.js
- TypeScript
- React
- Next.js
- Shadcn UI
- Runway API
- Supabase
